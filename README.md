🌟 Astra AI
Astra — Your intelligent voice-driven personal AI assistant.

🚀 Features
🎤 Wake word activation ("Astra")
🔐 Speaker authentication
🧠 Agentic AI (task execution)
🌐 Web search + 📚 RAG memory
🤖 Local LLM (Ollama)
🔊 Human-like voice output
🖼️ Face UI

🧠 Architecture
🎤 Wake Word ("Astra")
   ↓
🎧 Audio Input
   ↓
🧬 Speaker Verification
   ↓
🧠 Agent
   ↓
🤖 LLM (Ollama)
   ↓
🔊 Text-to-Speech
   ↓
🖼️ UI Response

🎤 Example
“Astra, open YouTube”
